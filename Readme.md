## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/airbnb-online-experiences-clone.git
```

Go to the project directory

```bash
  cd ..airbnb-online-experiences-clone
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
